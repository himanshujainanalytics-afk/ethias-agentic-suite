AWS Service Stack
Edge & CDN
CloudFront + Global Accelerator
450+ edge PoPs worldwide. Routes each advisor to nearest healthy region. <50ms latency target globally.
Auth & SSO
Cognito + IAM Identity Center
SAML/OIDC SSO for enterprise advisors. Built-in TOTP 2FA matching the login screen. Session management.
API Layer
API Gateway + AppSync
REST for CRM/portfolio endpoints. WebSocket via AppSync for live compliance scanner and inbox sentiment.
Compute
ECS Fargate
Serverless containers. Auto-scales 0→∞ per request volume. No server management. Pay-per-task-second.
Primary Database
Aurora PostgreSQL Global DB
Write region: us-east-1. Read replicas: eu-west-1, ap-southeast-1. <1s cross-region replication lag. 128TB auto-scale.
Cache & Sessions
ElastiCache for Redis
Market data cache (15-min TTL). AI insight caching. Session tokens. Cluster mode with auto-shard rebalancing.
File Storage
S3 + Intelligent-Tiering
AI voice training samples, audit PDF packages, firm logos. Lifecycle rules auto-archive to Glacier after 90 days.
AI Platform
Bedrock + SageMaker
Bedrock (Claude) for compliance ghostwriter, AI drafts, daily briefing. SageMaker endpoints for churn and propensity models.
Async Events
EventBridge + SQS
Triggers: AI briefing generation at 6 AM per timezone. Compliance scans on message send. TLH alerts on market close.
Calendar Sync
Lambda + Step Functions
Lambda polls Microsoft Graph API (Outlook) and Google Calendar API every 5 min. Step Functions orchestrate OAuth refresh flows.
Compliance & Audit
CloudTrail + Macie + Security Hub
Immutable audit log of every API call. Macie scans for PII leakage in S3. Security Hub maps to FINRA/SEC control frameworks.
Observability
CloudWatch + X-Ray
Distributed tracing across AI model calls, DB queries, and Lambda. Dashboards per advisory region. SLA alerting on p99 latency.

Auto-Scaling Strategy

⚡ Compute (ECS Fargate)

  • Scale out at 70% average CPU across task cluster
  • Min 2 tasks per region, max 200 during market open
  • Scheduled scale-up at 9:00 AM local per timezone
  • Scale-in cooldown: 5 min to avoid thrash
  • Zero cold-start — always-warm baseline containers

🗄️ Database (Aurora Global)

  • Storage auto-scales 10 GB → 128 TB, no downtime
  • Read replicas auto-added during peak trading hours
  • Regional failover: RPO < 1s, RTO < 1 min
  • Connection pooling via RDS Proxy (prevents spike exhaustion)
  • Automated snapshots every 5 min, retained 35 days

🧠 AI Layer (Bedrock + SageMaker)

  • Bedrock: serverless, no provisioning — auto-scales per token
  • SageMaker churn/propensity: async inference endpoints
  • Daily briefing batch jobs run 5:30 AM per timezone via EventBridge
  • ElastiCache caches AI outputs 30 min (reduces Bedrock cost ~60%)
  • Model drift monitoring via SageMaker Model Monitor
